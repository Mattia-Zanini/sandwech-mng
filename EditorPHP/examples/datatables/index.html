Editor example Embedded DataTable

Editor can use a DataTable as an input field or selection of one or multiple items from a list while taking advantage of all the advantage features DataTables offers. For example users can easily search for the item they are looking for from a large list. Server-side processing can also be used to make large data sets super responsive.

This set of examples shows how the datetable field type can be used in Editor.
